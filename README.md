# News-Website-Project
This is a simple and responsive News Website developed using HTML, CSS, and JavaScript. <br>
The website fetches and displays real-time news articles using an external API, providing users with the latest updates in an interactive and user-friendly interface.
<br>
Features
• Fetches real-time news using API
• Search functionality to find specific news
• Login and Sign-up UI (frontend design)
• About section
• Contact section
• Feedback form
• Responsive design using Bootstrap
<br>

Technologies Used
• HTML
• CSS
• JavaScript
• Bootstrap
• News API (NewsAPI.org)

<br>

How It Works
• The website connects to a news API to fetch the latest articles.
• JavaScript dynamically updates the page with news data.
• Users can search for specific topics using the search bar.
• Static sections like About, Contact, and Feedback enhance usability.

<br>
