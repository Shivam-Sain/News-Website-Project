# News-Website-Project
This is a simple and responsive News Website developed using HTML, CSS, and JavaScript. <br>
The website fetches and displays real-time news articles using an external API, providing users with the latest updates in an interactive and user-friendly interface.
<br>

Features <br>
• Fetches real-time news using API <br>
• Search functionality to find specific news<br>
• Login and Sign-up UI (frontend design)<br>
• About section<br>
• Contact section<br>
• Feedback form<br>
• Responsive design using Bootstrap
<br>

Technologies Used<br>
• HTML<br>
• CSS<br>
• JavaScript<br>
• Bootstrap<br>
• News API (NewsAPI.org)

<br>

How It Works<br>
• The website connects to a news API to fetch the latest articles.<br>
• JavaScript dynamically updates the page with news data.<br>
• Users can search for specific topics using the search bar.<br>
• Static sections like About, Contact, and Feedback enhance usability.

<br>
